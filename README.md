
# campus_social 校园交友系统

基于 C++ Socket 实现的校园交友系统，支持学号登录、一对一聊天、好友匹配邀请，打造无尴尬社交体验。

## 核心功能
- **学号登录**：使用学号作为唯一身份标识，快速登录
- **一对一聊天**：通过 `@学号:消息内容` 格式，实现点对点文字聊天
- **好友匹配邀请**：
  - 发送邀请：`@目标学号:MATCH`
  - 同意邀请：`@邀请者学号:ACCEPT`
  - 拒绝邀请：`@邀请者学号:REJECT`
- **多用户在线**：多线程服务端，支持多客户端同时连接
- **在线状态检测**：自动提示对方是否在线，避免消息发送失败

## 技术栈
- 语言：C++
- 网络：TCP Socket
- 并发：pthread 多线程
- 存储：内存存储（在线用户列表、好友关系）

## 编译与运行

### 1.编译

#### 使用 makefile
make
#### 删除编译产生的文件
make clean

### 2.运行
启动服务端:  
bash  
运行  
`./server`  
服务端将在 9527 端口监听客户端连接。  
启动客户端（可开多个终端模拟多用户）：  
bash  
运行  
`./client`  
输入学号完成登录后即可开始聊天 / 匹配。  
使用说明  

### 功能	指令格式	示例  
聊天	@目标学号:消息内容	@456:一起去图书馆吗？  
发送好友邀请	@目标学号:MATCH	@456:MATCH  
同意邀请	@邀请者学号:ACCEPT	@123:ACCEPT  
拒绝邀请	@邀请者学号:REJECT	@123:REJECT  
退出客户端	`quit`
